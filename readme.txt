== Description ==

**WPC Role-Based Shipping Methods** allow the limitation of available shipping methods for each user role individually. At checkout, customers will only see certain shipping options based on their role. For instance, an Admin will have access to all shipping methods, while a Guest will only see Local Pickup.

= Features =

- Manage user roles for all shipping methods on one screen
- Allow to select All, User (logged in), and Guest (non logged in)
- Allow to select all available user roles

== Installation ==

1. Please make sure that you installed WooCommerce
2. Go to plugins in your dashboard and select "Add New"
3. Search for "WPC Role-Based Shipping Methods", Install & Activate it
4. Now you can go to WP-admin >> Role-Based Shipping Methods to configure
